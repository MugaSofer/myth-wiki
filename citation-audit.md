- `apollodorus-frazer.txt` (Bibliotheca, Frazer translation)

### Methodology

For each citation:
1. Read the article claim and its associated footnote quote
2. Checked that the article's prose accurately reflects what the footnote says (Claim OK)
3. Searched the original source text file using grep for distinctive phrases from the footnote quote (Quote OK)
4. When grep failed due to line-wrapping in source files, used shorter phrases to confirm presence

### Limitations

The audit spanned two sessions with a context compaction in between. Phase 1 results (entries 1-74) were checked live but not written to this file before compaction occurred, so the per-entry detail was reconstructed afterward from the compaction summary. The summary reliably recorded which files were checked and that all citations passed, but the specific footnote IDs and notes for those entries are best-effort rather than exact. Phase 2 results (entries 75-100) have full grep evidence in the session log.
